# Foobar
Foobar is a python library

# Installation

use the package manager [pip]() to install foobar
```
pipe install foobar
```
# Usage
```
import foobar

foobar.pluralize('word') # returns 'words'
foobar.pluralize('goose') # returns 'geese'
foobar.singularize('phenomena') # returns 'phenomenon'
```
